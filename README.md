# 第三轮视觉方案预览（3 套 · 纯静态 · 可直接双击打开）

按你这次的反馈做的对比版。直接**双击**任意 `index.html` 即可在浏览器查看，**不需要启动服务器**。

```
审美优化/方案预览-第三轮/
├── README.md
├── 宣纸朱墨/index.html      ← 你最认可的方向（升级了动效）
├── 水墨燕园/index.html      ← 第一轮那套补齐成同规格，方便和宣纸朱墨并排对比
└── 蓝绿松石/index.html      ← 保留松石青主色，调整配色协调度（v2）
```

## 这一轮的三个重点

1. **回声纸条：新增「纸团展开 / 揉成一团」动效**
   - 第 4 屏「毕业回声」现在：出现＝纸团从一团展开成纸条；收起／点「收入百宝箱」＝纸条揉成一团抛走。
   - 可点下方按钮**实时重播**「纸团展开」「揉成一团」。
   - 借鉴技法：SVG `feTurbulence + feDisplacementMap` 置换滤镜做褶皱纹理 + transform（scale/rotate/圆角）把纸揉成团再展开。**自包含实现、无第三方库**。

2. **赛博礼物：新增烟花动效**
   - 第 10 屏「赛博礼物」出现时绽放多点烟花，可点按钮重播「出现 + 烟花」「离开」。
   - 借鉴技法：经典「径向粒子爆裂」——JS 生成粒子，CSS keyframes 让粒子按角度向外迸射 + 重力下坠 + 渐隐，多点错峰齐放。**自包含实现、无第三方库**。

   > 说明：我尝试联网检索 CodePen/文章作参考，但当前网络检索/抓取接口持续报错（503 / 域名被网络策略拦截），无法实时拉取页面。以上两类动画均为业内通用标准技法，我据此从零内联实现，效果与那些示例同源。

3. **三套对比 / 蓝绿松石调色**
   - 「宣纸朱墨」「水墨燕园」并排：都是水墨/朱砂方向，区别在水墨燕园用**冷调灰宣纸 + 走廊氤氲底图**，更画意；宣纸朱墨更暖、更克制。
   - 「蓝绿松石 v2」：保留你喜欢的松石青主色，**去掉原来对冲的互补橙**，改用一抹**克制暖铜**作唯一焦点（一等奖、转盘中心、票根边），冷青绿同类色统一画面，背景更安静——协调度明显提升。

## 每套仍是 11 屏

登录 / 主线答题 / 提交错误·冷却 / **回声·纸团动效** / 百宝箱·展开收起 / 主线完成 / 旁枝 / 终章·简谱 / 抽奖兑奖 / **赛博礼物·烟花** / 后台核销。

> 第 4、5、10 屏需浏览器开启 JavaScript 才能播放动效（默认都开），其余画面纯 CSS。
