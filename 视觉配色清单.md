# 蓝绿松石版 · 视觉配色清单

本文件整理 `static/styles.css`（及 `static/app.js` 中少量内联色）实际使用的全部颜色，
含**命名、HEX、RGB、使用位置**。配色取自 `设计稿参考_蓝绿松石.html`。

> 配色思路：**松石青为主色** + **克制暖铜为唯一焦点**（一等奖 / 转盘中心 / 票根边）
> + **柔青灰为中性**，冷青绿同类色统一画面，单一暖金属点睛，背景安静。错误态单独用暖陶红。

## 六、半透明遮罩 / 阴影（说明）

- **回声纸团遮罩**：`rgba(12,31,28,.92)`（深松石黑，12,31,28 @92%）+ 中心松石光晕 `rgba(19,138,125,.18)`。
- **赛博礼物遮罩**：`rgba(8,32,29,.95)`（8,32,29 @95%）+ 光晕 `rgba(19,138,125,.22)`。
- **普通弹层遮罩**：`rgba(11,31,28,.5)`。
- **阴影**统一以松石墨绿 `rgba(11,82,74,…)` 低透明度投射（如 `--shadow`、卡片/按钮投影），保持冷调一致；暖铜/松石的少量 `rgba` 描边用于聚焦元素（票根、转盘心、救援框）。

### 速查：三类主色

- **主色（松石青系）**：`#138a7d` / `#0e6f64` / `#0b524a` / `#d8ece8`

- **中性（青灰）**：`#93b3ac`；**错误**：`#c25b43`
